# Darsh_Hackpad
A 6-Key macropad designed to carry out common shortcuts quickly.

## Features:
- 3D printed case and cover
- 6x cherry-mx style keys
- 1x EC11 Rotary encoder
- 1x SK6812 MINI LEDs
- 1x Seeed XIAD RP2040
